# EasyMock改变调用 - EasyMock教程

EasyMock提供很多的方法来改变预期的调用计数。

*   times(int min, int max) - 最小值和最大值之间的预期调用。

*   atLeastOnce() - 预期至少有一个调用。

*   anyTimes() - 预期调用的数量不受限制。
